# Research

This folder is for active research work on the new direction — whatever that turns out to be.

**Current status:** Empty. Research direction not yet selected. Field survey in progress.

---

## Files (populate as work begins)

| File | Purpose |
|------|---------|
| `survey.md` | Field survey results — what's being done in music AI / MIR, open problems |
| `papers.md` | Reading notes and citations — one entry per paper |
| `experiments.md` | Experiment log — what was run, setup, results, interpretation |

---

## How to Add a Paper Entry (`papers.md`)

```markdown
## [Paper Title]
**Authors:** [names] | **Year:** [year] | **Venue:** [ISMIR / NeurIPS / etc.]
**Link:** [URL or DOI]

**What it does:** [1-2 sentence summary]
**Relevance:** [why it matters for current work]
**Key finding:** [the one thing to remember]
**Code available:** Yes / No / [link]
```

## How to Add an Experiment Entry (`experiments.md`)

```markdown
## [Experiment Name] — [Date]
**Goal:** [what you were testing]
**Setup:** [dataset, model, hyperparameters, hardware]
**Results:** [numbers, metrics, observations]
**Interpretation:** [what it means]
**Next step:** [what to do based on this result]
```
