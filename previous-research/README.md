# Previous Research

This folder contains archived work from the TriMET and TriMET-drum projects — the research direction pursued before the current MTR trajectory.

**None of this work is wasted.** The technical skills built (signal processing, audio analysis, MIDI processing, ML pipeline design), the dataset knowledge (Groove MIDI Dataset, Lakh MIDI, POP909), and the conceptual frameworks developed (four-axis model of musical information) are all transferable to new directions.

## How to Use This Folder

- **For context on prior technical work:** read `trimet-drum.md` — it has the most current and accurate account of what was actually built and what was found
- **For the original broader vision:** read `trimet-overview.md`
- **For understanding the transition:** read `why-we-shifted.md`
- **For Claude:** this context is already in the Claude Project memory — you do not need to re-read these files in every session
- **For ChatGPT:** if a task requires understanding prior work, read `trimet-drum.md` first
